# A Guide on how to edit the Assignment/Experiment

## Headings
### H1 Tag
#### Markdown Syntax 1:
```
# Text
```
#### Markdown Syntax 2:
```
Text
====
```
#### HTML Syntax:
```
<h1>Text</h1>
```
### H2 Tag
#### Markdown Syntax 1:
```
## Text
```
#### Markdown Syntax 2:
```
Text
----
```
#### HTML Syntax:
```
<h2>Text</h2>
```
### H3 Tag
#### Markdown Syntax:
```
### Text
```
#### HTML Syntax:
```
<h3>Text</h3>
```
### H4 Tag
#### Markdown Syntax:
```
#### Text
```
#### HTML Syntax:
```

```
### H5 Tag

```

```
#### HTML Syntax:

```
### H6 Tag
#### Markdown Syntax:
```

```
#### HTML Syntax:

```

## Text Highlighting
### Bold/Strong
#### Markdown Syntax:
```
**Text**
```
#### HTML Syntax 1:
```
<b>Text</b>
```
#### HTML Syntax 2:
```
<strong>Text</strong>
```
### Italics/Emphasis
#### Markdown Syntax 1:
```
*Text*
```
#### Markdown Syntax 2:
```
_Text_
```
#### HTML Syntax 1:
```
<i>Text</i>
```
#### HTML Syntax 2:
```
<em>Text</em>
```

## Blockquote
### Normal Blockquote
#### Markdown Syntax:
```
> Text
```
### Multi-paragraph Blockquote
#### Markdown Syntax:
```
> Text 1
>
> Text 2
```
### Nested Blockquote
#### Markdown Syntax:
```
> Text 
>> Nested Text
```
